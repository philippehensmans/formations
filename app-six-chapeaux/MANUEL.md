# Six Chapeaux de Bono - Manuel d'utilisation

## Description

Cette application permet de recueillir les avis des participants selon la methode des **Six Chapeaux de la Reflexion** d'Edward de Bono. Chaque participant peut exprimer plusieurs avis, chacun associe a un chapeau representant un mode de pensee different.

## Les Six Chapeaux

| Chapeau | Couleur | Mode de pensee |
|---------|---------|----------------|
| Blanc | Gris/Blanc | Faits, donnees, informations objectives |
| Rouge | Rouge | Emotions, sentiments, intuitions |
| Noir | Noir | Prudence, critique, risques, problemes |
| Jaune | Jaune | Optimisme, avantages, aspects positifs |
| Vert | Vert | Creativite, nouvelles idees, alternatives |
| Bleu | Bleu | Organisation, controle du processus, synthese |

## Pour les Participants

### Connexion
1. Accedez a l'application via le lien fourni par le formateur
2. Entrez le **code de session** fourni (ex: ABC123)
3. Connectez-vous avec vos identifiants ou inscrivez-vous

### Ajouter un avis
1. Cliquez sur l'un des six chapeaux correspondant a votre type de reflexion
2. Saisissez votre avis dans le champ de texte
3. Cliquez sur **Enregistrer**
4. Vous pouvez ajouter autant d'avis que vous souhaitez

### Gerer vos avis
- **Modifier** : Cliquez sur l'icone crayon a cote de votre avis
- **Supprimer** : Cliquez sur l'icone poubelle
- **Partager** : Cliquez sur "Partager tous mes avis" pour les rendre visibles au formateur

## Pour les Formateurs

### Creer une session
1. Connectez-vous sur la page Formateur
2. Entrez un nom pour votre session
3. Notez le **code de session** genere a partager avec les participants

### Definir le sujet de reflexion
1. Depuis la vue session (cliquez sur "Voir tout")
2. Cliquez sur **Modifier** a cote du sujet de reflexion
3. Entrez la question ou le theme sur lequel les participants doivent reflechir

### Visualiser les resultats

#### Vue individuelle
- Cliquez sur un participant pour voir tous ses avis classes par chapeau

#### Vue globale (session-view)
- Cliquez sur **Voir tout** depuis le tableau de bord formateur
- Trois modes d'affichage disponibles :
  - **Grille par chapeau** : Tous les avis regroupes par type de chapeau
  - **Liste chronologique** : Tous les avis tries par date
  - **Par participant** : Avis regroupes par personne

#### Filtrer les resultats
- Cliquez sur un chapeau pour filtrer uniquement les avis de ce type
- Les statistiques en haut de page montrent la repartition globale

### Exporter / Imprimer
- Utilisez le bouton **Imprimer** pour generer une version imprimable des resultats

## Conseils d'animation

1. **Expliquez la methode** : Prenez le temps de presenter chaque chapeau avant de commencer
2. **Posez une question claire** : Definissez un sujet de reflexion precis
3. **Encouragez la diversite** : Invitez les participants a utiliser differents chapeaux
4. **Exploitez les resultats** : Utilisez la vue filtree pour discuter chapeau par chapeau

## Support technique

En cas de probleme, contactez le formateur ou l'administrateur de la plateforme.
